Get properties

val get : t -> string -> 'a decoder -> 'a

get obj prop decoder is the value of the property prop in obj. If prop is undefined in obj, Undefined_property is raised.

Note: prop must be an ASCII string.

val get_opt : t -> string -> 'a decoder -> 'a option

get_opt obj prop decoder is the value of the property prop in obj if prop is defined, and None otherwise.

Note: prop must be an ASCII string.
